Это премиальная миниапп-версия под Telegram.
Что делать:
1. Распакуй архив.
2. Залей все файлы в свой GitHub репозиторий вместо старых.
3. Pages уже можно использовать как сайт.
4. Для Telegram Mini App укажи ссылку через BotFather /setmenubutton.
5. В app.js потом вставишь свой API_URL и сменишь USE_MOCK на false.

Фронт ждёт действия:
- dashboard
- analyze_food
- save_food
- update_weight
- profile_analysis
