Съел? Считай! — тестовая PWA-версия

Как запустить:
1. Распакуй архив.
2. Открой папку в VS Code.
3. Запусти любой локальный сервер, например:
   - Python: python3 -m http.server 8080
4. Открой в Safari на iPhone адрес сервера.
5. Нажми Поделиться -> На экран Домой.

Что внутри:
- PWA с экранами: главная, еда, день, неделя, профиль, вес, анализ.
- Локальное хранение данных в LocalStorage.
- Тестовый расчёт еды по простым правилам.
- Фото-сценарий пока как UX-заглушка; для боевой версии нужно подключить твой n8n API.

- мой день / неделя / вес
